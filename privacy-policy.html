Privacy Policy

Last updated: August 22, 2025

Summary: This extension operates entirely locally on your device. No personal data is transmitted to external servers or third parties.

1. Information We Collect

The Crowley Marine Cache Cleaner extension collects minimal usage statistics for display purposes only:

  • URL Count: Number of Crowley Marine URLs that have been modified
  • Timestamps: When the extension was last used
  • URL History: Last 10 URLs visited on crowleymarine.com (stored locally only)
  • Daily Statistics: Count of pages cleared per day

2. How We Use Information

The collected information is used exclusively to:

  • Display usage statistics in the extension popup
  • Show the last URL that was modified
  • Track daily and total usage counts
  • Provide user feedback on extension activity

3. Data Storage and Security

All data is stored locally on your device using Chrome's secure storage API:

  • Local Storage Only: No data is transmitted to external servers
  • Chrome Storage API: Uses Chrome's built-in secure storage mechanism
  • No External Access: Data cannot be accessed by other websites or extensions
  • User Control: Data is automatically removed when the extension is uninstalled

4. Data Sharing

We do not share, sell, or transmit any data to third parties. All information remains on your local device.

5. Data Retention

  • URL History: Limited to the last 10 URLs visited
  • Daily Statistics: Retained indefinitely on your device
  • Total Counts: Retained until extension is uninstalled
  • User Control: All data is removed when you uninstall the extension

6. Permissions Explanation

The extension requests the following permissions:

  • storage: To save usage statistics locally on your device
  • activeTab: To access the current tab when you interact with the extension
  • *://*.crowleymarine.com/*: To modify URLs only on Crowley Marine websites

7. Third-Party Services

This extension does not use any third-party services, analytics, or tracking tools. It operates completely independently.

8. Changes to Privacy Policy

We may update this privacy policy from time to time. Any changes will be reflected in the extension and on this page with an updated "Last updated" date.

9. Your Rights

You have the right to:

  • Disable the extension at any time using the toggle in the popup
  • Uninstall the extension to remove all stored data
  • View all stored statistics in the extension popup

10. Contact Information

If you have any questions about this privacy policy or the extension's data practices, please contact us at:

Email: support@crowleycachecleaner.com
